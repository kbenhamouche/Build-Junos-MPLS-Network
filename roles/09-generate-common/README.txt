Role "Common" has the following purposes:
1- Generate a list of parameters shared by all nodes
2- The generate config are based on the defaults values in 'main.yaml' file.

The common configuration includes:
    Hostname
    Timezone
    Root password
    Login Banner
    Domain Name
    DNS servers
    NTP servers
    Netconf SSH
    SNMP
    Syslog
    Management Interface
    Static Route
    LLDP
